# 干支カレンダー (eto60cal)

## 概要

六十干支・九星を表示する年間カレンダーです。  
西暦を入力すると、その年の各月・各日の干支と九星を表示します。

## 機能

- 週カレンダー表示（7列形式）と月カレンダー表示の切り替え
- 九星・干支の縦表示
- 節入り日のハイライト表示（緑色）
- PC・スマートフォン対応（レスポンシブデザイン）
- 対応年: 1924年〜2033年

## 動作環境

- PHP 7.4 以上
- Apache（.htaccess 有効）

## 使い方

1. ブラウザで以下のURLにアクセス
   - https://adwat.org/eto60cal/
2. 西暦（1924〜2033）を入力して「SEND」をクリック
3. 週カレンダー／月カレンダーを切り替えて表示

## ファイル構成

- `eto60cal.php` — 干支カレンダー本体
- `output.css` — スタイルシート
- `.htaccess` — DirectoryIndex 設定
