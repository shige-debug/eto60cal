# 干支カレンダー / Eto60 Calendar (eto60cal)

[日本語](#日本語) | [English](#english)

---

## 日本語

### 概要

六十干支・九星を表示する年間カレンダーです。  
西暦を入力すると、その年の各月・各日の干支と九星を表示します。

### 機能

- 週カレンダー表示（7列形式）と月カレンダー表示の切り替え
- 九星・干支の縦表示
- 節入り日のハイライト表示（緑色）
- PC・スマートフォン対応（レスポンシブデザイン）
- 対応年: 1924年〜2033年

### 動作環境

- PHP 7.4 以上
- Apache（.htaccess 有効）

### 使い方

1. ブラウザで以下のURLにアクセス
   - https://adwat.org/eto60cal/
2. 西暦（1924〜2033）を入力して「SEND」をクリック
3. 週カレンダー／月カレンダーを切り替えて表示

### ファイル構成

- `eto60cal.php` — 干支カレンダー本体
- `output.css` — スタイルシート
- `.htaccess` — DirectoryIndex 設定

## English

### Overview

A yearly calendar displaying the 60-cycle Eto (Chinese zodiac) and Nine Stars (Kyusei) for each day.  
Enter a year to view the Eto and Nine Star for every month and day of that year.

### Features

- Switch between weekly calendar (7-column layout) and monthly calendar view
- Vertical display of Nine Stars and Eto
- Highlighted Setsuiri days (solar term entry days) in green
- Responsive design for PC and smartphones
- Supported years: 1924–2033

### Requirements

- PHP 7.4 or later
- Apache (with .htaccess enabled)

### Usage

1. Open the URL in your browser:
   - https://adwat.org/eto60cal/
2. Enter a year (1924–2033) and click SEND
3. Toggle between weekly and monthly calendar views

### File Structure

- `eto60cal.php` — Main calendar application
- `output.css` — Stylesheet
- `.htaccess` — DirectoryIndex configuration
